CREATE TABLE pelanggan (
    id INT AUTO_INCREMENT PRIMARY KEY,
    nama VARCHAR(100),
    nominal INT,
    jenis_cuci VARCHAR(100),
    berat FLOAT,
    tanggal DATE
);

CREATE TABLE dataset_intent (
    id INT AUTO_INCREMENT PRIMARY KEY,
    kalimat VARCHAR(255),
    intent VARCHAR(255)
);


CONTOH QUERY 
SELECT COUNT(*) AS jumlah_cuci 
FROM pelanggan 
WHERE nama = "indah" 
AND MONTH(tanggal) = 10
AND YEAR(tanggal) = 2025;

SELECT 
    nama,
    COUNT(*) AS jumlah_cuci,
    SUM(nominal) AS total_nominal
FROM pelanggan
WHERE MONTH(tanggal) = 3
AND YEAR(tanggal) = 2025
GROUP BY nama
ORDER BY jumlah_cuci DESC, total_nominal DESC;

================ untuk  entity extract 

Pertanyaan bagus. Jawabannya:

**kata seperti "siapa", "apa", "kenapa", "berapa" itu BUKAN entity.**

Mereka disebut:

# Question Word / Interrogative Word

atau dalam NLP disebut:

# Intent Indicator / Query Word

---

# 1. Contoh kalimat kamu:

Kalimat:

```text
siapa pelanggan terbaik bulan ini
```

Breakdown yang benar:

| Kata      | Label     |
| --------- | --------- |
| siapa     | QUESTION  |
| pelanggan | OBJECT    |
| terbaik   | STATUS    |
| bulan     | TIME_UNIT |
| ini       | TIME_REF  |

---

# 2. Kenapa "siapa" bukan entity utama?

Karena:

"siapa" tidak punya nilai data

dia hanya menunjukkan jenis pertanyaan

Contoh:

Kalau sistem jawab:

```text
Irfan
```

"Irfan" → entity PERSON

bukan "siapa"

---

# 3. Fungsi kata tanya dalam chatbot

Kata tanya membantu menentukan:

jenis jawaban

Contoh:

| Kata   | Jawaban    |
| ------ | ---------- |
| siapa  | nama orang |
| berapa | angka      |
| apa    | benda      |
| kapan  | waktu      |

---

# 4. Dalam sistem kamu

Naive Bayes sudah handle ini

Contoh:

```text
siapa pelanggan terbaik
```

intent:

```text
rekomendasi_pelanggan_terbaik
```

jadi kamu tidak perlu ekstrak "siapa"

---

# 5. Yang wajib kamu ekstrak hanya ini:

Untuk sistem laundry kamu:

| Entity        | Contoh     |
| ------------- | ---------- |
| STATUS        | terbaik    |
| TIME_NUMBER   | 3          |
| TIME_UNIT     | bulan      |
| TIME_REF      | ini        |
| METRIC        | pendapatan |
| CUSTOMER_TYPE | vip        |
| PERSON        | irfan      |

---

# 6. "berapa" juga sama

Kalimat:

```text
berapa total pendapatan bulan ini
```

Entity:

```text
METRIC = pendapatan
TIME_UNIT = bulan
TIME_REF = ini
```

"berapa" → tidak perlu disimpan

QUESTION_WORD = [
